# Abhi's 3D Portfolio

Futuristic portfolio website with Three.js, GSAP, and Framer Motion.

## 🚀 Deploy to GitHub Pages

### Step 1: Initialize Git Repository
```bash
cd "c:\Users\abhis\Downloads\Abhi Graphic 3D Portfolio"
git init
git add .
git commit -m "Initial commit: Futuristic 3D Portfolio"
```

### Step 2: Create GitHub Repository
1. Go to https://github.com/Abhitech8code
2. Click "New Repository"
3. Name it: `portfolio`
4. Don't initialize with README
5. Click "Create repository"

### Step 3: Push to GitHub
```bash
git remote add origin https://github.com/Abhitech8code/portfolio.git
git branch -M main
git push -u origin main
```

### Step 4: Enable GitHub Pages
1. Go to repository Settings
2. Click "Pages" in sidebar
3. Under "Build and deployment":
   - Source: GitHub Actions
4. Wait for deployment (check Actions tab)

### Step 5: Visit Your Site
Your site will be live at: `https://Abhitech8code.github.io/portfolio/`
